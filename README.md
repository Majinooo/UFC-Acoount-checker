A powerful, fast, and reliable account checker for UFC (Ultimate Fighting Championship) platforms. Efficiently verify your accounts using combo lists and proxy support.

---

## 🚀 Features

- **⚡ Multi-Threading:** High-speed verification using concurrent requests (threads).
- **🛡️ Proxy Support:** Supports HTTP/S, SOCKS4, and SOCKS5 proxies to bypass IP bans and rate-limiting.
- **📊 Clean CLI:** Modern and clean command-line interface (CLI) featuring real-time statistics (Hits, Bad, Free, Errors).
- **📁 Automated Saving:** Valid accounts (Hits) are automatically sorted and saved into separate text files inside the `results/` directory.
- **🔄 Auto-Retry:** Automatically retries failed requests caused by proxy instability or timeouts.
